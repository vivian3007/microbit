Over

De definitie van een micro:bit is "a pocket-sized computer that introduces you to how software and hardware work together" (The Micro:bit Educational Foundation, z.d.). De ontwikkeling van de micro:bit begon in 2014 door de BBC in het Verenigd Koninkrijk. Het doel van deze micro:bit is om programmeren wat toegankelijker te maken. Met de micro:bit lag de focus vooral op het onderwijs, om kinderen al vroeg te stimuleren om creatief en gemakkelijk om te kunnen gaan met elektronica.

In 2016 hebben leerlingen daarom in Wales, Noord-Ierland en Schotland met een leeftijd van 11 of 12 jaar een micro:bit ontvangen om daarmee te leren hoe ze kunnen programmeren.

Deze micro:bit is ontzettend goed ontvangen en tegenwoordig is het apparaat heel populair geworden. Je kan wel zeggen dat de micro:bit het doel ruim behaald heeft, omdat kinderen over de hele wereld tegenwoordig leren omgaan met de micro:bit.

De micro:bit is na 2016 nog wel aangepast en heeft meer functies gekregen, wat ervoor zorgde dat er nog meer mogelijk was met dit kleine computertje.

De micro:bit kan je via een USB kabel aansluiten aan je computer en op die manier kan je je code erop zetten. Over de code volgt later meer.

Input

De definitie van een input is: "Elke soort middel dat nodig is om een proces op te starten: bijv. de noden van de gebruikers, financiële, menselijke en technologische middelen" (Input Nederlands Woordenboek - Woorden.Org, z.d.). De Micro:bit heeft verschillende inputs waarmee je uiteindelijk van alles kan programmeren. De inputs die op de micro:bit zelf zitten zijn:

  • Buttons – De micro:bit heeft 2 buttons, de A button en de B button. Deze buttons kan je zo programmeren dat er iets gebeurt als je op deze knoppen klikt.
  • Accelerometer – Dit is een soort bewegingssensor op de micro:bit die het kan waarnemen als de micro:bit wordt bijvoorbeeld geschud, als hij valt of als hij ondersteboven ligt.
  • Temperatuursensor – De micro:bit heeft een temperatuursensor die de temperatuur van de lucht kan meten.
  • Lichtsensor – De micro:bit heeft een lichtsensor die via de LED’s op het bordje ook het lichtniveau kan meten.
  • Compass – Deze sensor detecteert magnetische velden. Daardoor weet hij welke kant de micro:bit op wijst.
  • Touch sensor – De micro:bit heeft een touch sensor, die bij aanraking ook als input kan dienen. Hij werkt wel alleen met de aanraking van je vinger. Deze input zit alleen op nieuwe versies van de micro:bit.
  • Microfoon – De micro:bit heeft een ingebouwde microfoon die geluid kan detecteren. Deze input zit alleen op nieuwe versies van de micro:bit.
  • Externe input – Naast alle inputs die op de micro:bit zelf zitten kan je er ook externe inputs aan vast maken via de pins aan de onderkant. Dit kan bijvoorbeeld een externe knop of een touchsensor zijn.

Output

De definitie van een output is: "Uitvoer van een programma, bijvoorbeeld naar een bestand, de printer of het scherm" (Output Nederlands Woordenboek - Woorden.Org, z.d.). Er zijn ook verschillende outputs die op de micro:bit zitten:

  • LED’s – De micro:bit heeft 25 LED lichtjes waarmee allemaal verschillende vormen en patronen gemaakt kunnen worden door code te schrijven.
  • Radio – De micro:bit kan radiogolven gebruiken om met een andere micro:bit te communiceren. Zo kan je boodschappen versturen en ontvangen.
  • Sound/speaker – De micro:bit heeft een ingebouwde speaker waarmee je muziek en geluid kan laten afspelen. Je kan via de pins zelfs muziek luisteren via je koptelefoon of oortjes. De speaker output zit alleen op de nieuwe versies van de micro:bit. De sound output zat ook op de oude alleen daar moest je dan een extern apparaat aansluiten op de pins.
  • Externe output – Naast alle outputs die op de micro:bit zelf zitten kan je er via de pins ook externe outputs op aansluiten. Dit kan bijvoorbeeld dus een speakertje zijn.

Code

Voordat de micro:bit iets kan doen moet er code op gezet worden die de micro:bit vertelt wat hij moet gaan doen. In de code moet altijd een input en output gecombineerd worden, omdat er anders niks gebeurt.

De code die nodig is voor de micro:bit kan je zelf maken in Makecode. Dat is een site waar je aan de hand van blokken kan programmeren. Deze blokken zijn te zien op afbeelding 1. Door deze blokken code op een bepaalde manier in elkaar te schuiven kan je heel makkelijk de microbit programmeren om iets te doen.

Deze blokken hebben natuurlijk geen eindeloze mogelijkheden, maar je kan de micro:bit ook programmeren met Javascript of Python. Dat zijn 2 codetalen waarbij je dus zelf je code moet typen, maar op die manier heb je wel meer mogelijkheden tijdens het programmeren.

Als de code voor de micro:bit is geschreven kan je die via een USB-kabel op de micro:bit zetten en dan zou dit moeten werken.

Robot

Voor het vak CLE ben ik een robot aan het maken (zie afbeelding 2). Ik heb nu een prototype van karton gemaakt die ik heb geprogrammeerd met de Circuit Playground Express (CPE). Dat is het bordje wat op afbeelding 3 te zien is. De circuit Playground Express lijkt eigenlijk best veel op de micro:bit. Het is eigenlijk een iets uitgebreidere versie ervan. De CPE heeft LED lichtjes die alle mogelijke kleur kunnen krijgen, 7 touch sensors in plaats van maar 1, meer pins en nog een aantal andere extra ‘s. Verder zijn de boards redelijk gelijk aan elkaar.

Zelf vond ik de micro:bit iets makkelijker om mee te werken, omdat deze iets minder opties heeft dan de CPE. Ik denk dat projecten opstarten met de micro:bit een super leuke manier is om kinderen van jongs af aan al een beetje te leren programmeren.
